Frequently Asked Questions

Quick answers to common questions before submitting a request.

How do I download mods from your recommendations?

▾
Each mod card on the Minecraft Mods page has a "Download" button that links directly to the official source (Modrinth, CurseForge, or the creator's site). Click it and follow the download instructions on that page.

Can I hire you for a Minecraft build or website?

▾
Yes! You can use the "Hire Me" button on the Skills page or submit a request through the form below with details about your project.

Which Minecraft versions do you support?

▾
Most of my mod recommendations target the latest stable releases (1.20+). However, some mods may support older versions — always check the version compatibility on the download page.

How long does it take to get a response?

▾
I typically respond within 24–48 hours. During busy periods it may take slightly longer, but every request is reviewed and answered.

Do you offer server setup services?

▾
Yes, I can help with Spigot/Paper server configuration, plugin setup, and world management. Submit a support request below with your server details and requirements.
